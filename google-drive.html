Google Drive

Send backup files to Google Drive.

Options?

    • client_id
      Type: string
      Client id.
    • client_secret
      Type: string
      Client secret
    • auth_code
      Type: string
      Auth code
    • token_file
      Type: string
      File to write all access token info after we get it from Google so we can use it in the future with refresh token.

Example 1

$backup = new MaratusBackup();

$gDriveConfig = array(
    'type' => 'gdrive',
    'client_id' => '112121212-12121212126.apps.googleusercontent.com',
    'client_secret' => 'xxxxxxxxxxxxccccccccc',
    'token_file' => 'gdrive-token.json',
    'auth_code' => '345ewsfssdfdsfs',
);
$backup->addStorage($gDriveConfig);

/**
 *  .... rest of the code to archive etc...
 */

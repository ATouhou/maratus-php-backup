Barracuda Copy.com

Send backup files to Copy.com storage.

Options?

    • consumer_key
      Type: string
      Consumer key.
    • consumer_secret
      Type: string
      Consumer Secret
    • access_token
      Type: string
      Access token
    • token_secret
      Type: string
      Token secret.

Example 1

$backup = new MaratusBackup();

$copyConfig = array(
    'type' => 'copy',
    'consumer_key' => 'xxxxxxxxxxxxxxxxxx',
    'consumer_secret' => 'yyyyyyyyyyyyyyyy',
    'access_token' => 'zzzzzzzzzzzzzzzzzz',
    'token_secret' => 'qqqqqqqqqqqqqqqq',
);

$backup->addStorage($copyConfig);

/**
 *  .... rest of the code to archive etc...
 */

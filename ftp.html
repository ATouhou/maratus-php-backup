FTP

Send backup files to FTP server.

Options?

    • host
      Type: string
      FTP host
    • port
      Type: int
      FTP port
    • user
      Type: string
      FTP username
    • pass
      Type: string
      Consumer Secret
    • remote_dir
      Type: string
      Remote folder to store files in.

Example 1

$backup = new MaratusBackup();

$ftpStorage = array(
    'type' => 'ftp',
    'host' => '192.168.1.2',
    'user' => 'test',
    'pass' => 'test',
    'remote_dir' => 'test2',
);
$backup->addStorage($ftpStorage);

/**
 *  .... rest of the code to archive etc...
 */
